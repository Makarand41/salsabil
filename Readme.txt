This project was developed as the final internship assignment at Nexus Info. It showcases a professional website for a software company, designed to highlight services, portfolio, and innovation in the IT sector.

🌐 Key Features

Responsive Design: Optimized for all screen sizes

Interactive UI: Smooth navigation and dynamic content

Portfolio Section: Displays projects and case studies

Service Details: Describes the company’s offerings

🛠️ Technologies Used

HTML5

CSS3 (with SCSS)

JavaScript

📁 Project Structure

index.html: Main homepage

style.scss & style.css: Styling files

script.js: Adds interactivity

🚀 Setup Instructions

Clone the repository

Navigate to the project folder

Open index.html in a browser to view the site
